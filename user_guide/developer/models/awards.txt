get_all_awards()

Parameters		=>	the order to display items (*asc*, desc, random)
					the display flag to use (*y*, n, all)
Returns			=>	query object
Description		=>	pulls all awards from the database

==========

get_award()

Parameters		=>	the award ID from the database
Returns			=>	query object
Description		=>	retrieves the award from the database

==========

get_awardees()

Parameters		=>	the award ID from the database
Returns			=>	query object
Description		=>	retrieves everyone who's been awarded a specific award

==========

get_player_awards()

Parameters		=>	player ID
					how many items should be returned (*10* - 0 will return all records)
Returns			=>	query object
Description		=>	retrieves all a player's awards, including OOC awards as well as all IC awards for all characters

==========

get_awards_for_id()

Parameters		=>	any numerical ID that matches an ID from either the players or characters tables
					a type (*character*, player)
Returns			=>	query object
Description		=>	retrieves all awards for a given player or character based on their ID
